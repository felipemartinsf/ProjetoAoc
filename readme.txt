1º - https://github.com/felipemartinsf/ProjetoAoc (link do projeto no github)
2º - Agora,é necessário ter o node instalado no computador. (https://nodejs.org/en/download)
3º - Depois, ao abrir o terminal na pasta ProjetoAoc na sua IDE escolhida (usamos o Visual Studio Code), rodar o comando npm i (instala as dependências necessárias para hospedar o servidor web localmente)
4º - Por fim, executar o comando npm run dev para abrir o servidor (será mostrado um link que, ao abrir no seu computador, abrirá o site)
